# Blog Application

Basic Blog application built using MERN stack.

## Proved options for
- Registering new users
- Create new Blog
- Update Blog
- Delete Blog

## To do list
- Comments by readers
- Likes and dislikes



## Deployment

### Backend

To deploy this project run

```bash
  npm install
```

```bash
  npm run server
```

### Frontend

```bash
cd client
```

```bash
npm run dev
```
